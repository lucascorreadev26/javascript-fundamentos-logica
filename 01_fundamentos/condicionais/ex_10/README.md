# Exercício 09 - Maior de Três Números
## Problema
Compare três números e informe o maior.

## Entrada
5 - 10 - 15

## Saída esperada
O numero maior eh: 15

## Raciocínio
1. Criar variáveis para receber e armazenar os valores dos números.
2. Criar variáveis para receber os valores dos inputs e forçar a tipagem para Number.
3. Criar uma variavel do tipo let para receber o numero maior entre os 3 digitados pelo usuáruio.
4. Criar a primeira verificação entre num1 e num2 para identificar o maior entre eles.
5. Criar a segunda verificação entre num3 e maior para identificar o número maior absoluto. 
6. Informar qual dos 3 números é o maior absoluto.

## Conceitos usados
- Variáveis 
- Estado de variáveis 
- Métodos internos: prompt() / Number() / console.log()
- Funções  
- Condicionais: if else
