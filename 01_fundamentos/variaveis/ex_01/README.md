# Exercício 01 - Dizer "Olá Mundo" no console

## Problema
Enviar um "Olá Mundo" no console do navegador

## Raciocínio
1. Criar estrutura de pastas e arquivos
2. Criar uma variavel e armazenar o valor na variavel
3. Utilizar o método console.log nessa variavel

## Conceitos usados
- Variáveis
- Metodo console.log
