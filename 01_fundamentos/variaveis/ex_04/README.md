# Exercício 03 - Triplo de um número

## Problema
Receber um número e informar o triplo dele.

## Entrada
10 / 100 / 5000

## Saída esperada
30 / 300 / 15000

## Raciocínio
1. Criar variável para receber e armazenar o valor do número.
2. Criar variável para armazenar e calcular o triplo do número.
3. Utilizar o método console.log para exibir as informações no console do navegador.

## Conceitos usados
- Variáveis
- Método prompt
- Método console.log
