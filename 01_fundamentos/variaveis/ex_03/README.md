# Exercício 03 - Dobro de um número

## Problema
Receber um número e informar o dobro dele.

## Entrada
10 / 100 / 5000

## Saída esperada
20 / 200 / 10000

## Raciocínio
1. Criar variável para receber e armazenar o valor do número.
2. Criar variável para armazenar e calcular o dobro do número.
3. Utilizar o método console.log para exibir as informações no console do navegador.

## Conceitos usados
- Variáveis
- Método prompt
- Método console.log
