# Exercício 09 - Maior de Dois Números
## Problema
Compare dois números e informe o maior.

## Entrada
10 - 11 / 20 - 26 / 

## Saída esperada
O primeiro número é maior / O segundo número é maior

## Raciocínio
1. Criar variáveis para receber e armazenar os valores dos números.
2. Criar variáveis para receber os valors dos inputs e forçar a tipagem para Number.
3. Criar as condicionais para verificação do número.


## Conceitos usados
- Variáveis
- Métodos internos: prompt() / Number() / console.log()
- Condicionais: Operador ternário
- Funções
