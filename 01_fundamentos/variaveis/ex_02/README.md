# Exercício 02 - Variáveis básicas

## Problema
Criar variaveis para armazenar nome, idade e cidade de uma pessoa. Exibir essas informações no console.

## Raciocínio
1. Criar 3 variáveis para armazenar os respectividos dados.
2. Utilizar o método console.log para exibir as informações no console do navegador.

## Conceitos usados
- Variáveis
- Método console.log
- Template literal (opcional)
