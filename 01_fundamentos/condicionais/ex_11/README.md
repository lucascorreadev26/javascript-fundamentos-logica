# Exercício 11 - Aprovação por nota
## Problema
Receba 1 nota e informe se o aluno foi aprovado. Média = 7.

## Entrada
5 - 10 - 7

## Saída esperada
Reprovado / Aprovado / Aprovado

## Raciocínio
1. Criar variável para receber e armazenar o valor da nota.
2. Criar variável para receber o valor da nota e forçar a tipagem para Number.
3. Criar verificação da nota.
4. 1º verificação: Se a nota for menor que 7 - Reprovado
5. 2º verificação: Se a nota for maior que 7 - Aprovado

## Conceitos usados
- Variáveis 
- Métodos internos: prompt() / Number() / console.log()
- Funções  
- Condicionais: if else
